WHAT I TRIED — FULL REVIEWED MULTI-PAGE STATIC WEBSITE

**What I Tried** is a personal food discovery and review website featuring restaurants, cafés, dishes, ratings, favourites, must-try recommendations, and honest food experiences.

## Live Website

[View the website](https://krishnateja56.github.io/what_i_tried/)

## Features

- Explore restaurants, cafés, and food spots
- Browse dishes by category and city
- View top-rated and must-try recommendations
- Save favourite places
- Search restaurants and dishes
- Light and dark theme support
- Responsive design for mobile, tablet, and desktop
- Contact form interface
- Owner admin interface for managing restaurant data in the browser
- Local fallback images for selected restaurants.
